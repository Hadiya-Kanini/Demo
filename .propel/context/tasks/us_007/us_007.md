## Story ID
   * ID: US_007

## Story Title
   * Dataset presets & run configuration UI

## Description
  * As a presenter, I want dataset presets and a configuration UI so that I can quickly switch demo scenarios and adjust simulation parameters.

## Acceptance Criteria
  1. Given dataset presets exist, When a user opens the selector, Then at least three presets (Small Toy, Medium, Noisy) are available with brief descriptions.
  2. Given a preset is chosen while simulation is running, When user confirms via modal, Then simulation parameters update for subsequent runs and a dataset_selected event is emitted and logged.
  3. Given a user modifies configuration parameters manually, When they save, Then changes apply to future runs and persist if "Remember settings" is enabled.

## Edge Cases
   * How to validate invalid parameter ranges? (Show inline validation and prevent save.)
   * What if preset mapping is missing? (Fallback to safe default preset and log warning.)
