## Story ID
   * ID: US_011

## Story Title
   * Deterministic narrative messaging toggle & templates

## Description
  * As a presenter, I want an opt-in deterministic narrative messaging feature so that short contextual one-sentence summaries can aid storytelling without external models.

## Acceptance Criteria
  1. Given narrative toggle is off, When simulation runs, Then no narrative messages are shown and no external requests are made.
  2. Given narrative toggle is on, When an epoch completes, Then a single-sentence deterministic message is generated using local templates (e.g., "Epoch X: loss decreased by Y% — progress steady") and displayed in the UI.
  3. Given narrative messages, When user requests copy, Then the exact sentence is copied and logged; narratives are limited to one sentence and sanitized for PII.

## Edge Cases
   * What if template inputs are missing? (Skip narrative generation for that epoch and log the incident.)
   * If external model integration later added, ensure explicit consent flow and separate opt-in.
