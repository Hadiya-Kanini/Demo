## Story ID
   * ID: US_006

## Story Title
   * Event log with export/copy capability

## Description
  * As a presenter, I want a chronological event log with export and copy-to-clipboard so that I can share session artifacts and investigate behavior.

## Acceptance Criteria
  1. Given events are emitted, When an event occurs, Then the log appends a timestamped entry with message and metadata and visible in the LogList.
  2. Given the user clicks Export, When export executes, Then a downloadable JSON file containing the visible log entries is offered within 5 seconds.
  3. Given the user clicks Copy, When action completes, Then clipboard contains a human-readable formatted text of selected entries and an accessible confirmation is shown.

## Edge Cases
   * What if export fails due to storage limits? (Show meaningful error and offer partial export.)
   * How to handle very large logs? (Virtualize the list and provide "Export last N entries" option.)
